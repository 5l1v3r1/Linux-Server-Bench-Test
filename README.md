# VPSTEST.sh
VPS一键测试脚本，方便你更好地了解你的服务器，为[VPS.BEST](https://vps.best/70)量身定制，所有数据全部本地储存，无任何上传行为。
## 运行（不含UnixBench）
在 Terminal 中执行如下命令:
```
wget https://raw.githubusercontent.com/H2YTech/vpstest/master/vpstest.sh -N --no-check-certificate && bash vpstest.sh
```
## 运行（含UnixBench）
在 Terminal 中执行如下命令:
```
wget https://raw.githubusercontent.com/H2YTech/vpstest/master/vpstest.sh -N --no-check-certificate && bash vpstest.sh a
```
